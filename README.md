# htdocs Project

PHP プロジェクトのリポジトリです。

## 構成

-   `src/` - ソースコード
    -   `Application/` - アプリケーション層
    -   `Config/` - 設定ファイル
    -   `Controllers/` - コントローラー
    -   `Domain/` - ドメイン層
    -   `Infrastructure/` - インフラストラクチャ層
    -   `public/` - 公開ディレクトリ
-   `index.php` - エントリーポイント

## セットアップ

1. リポジトリをクローン
2. 必要に応じて依存関係をインストール
3. 環境設定を行う

## 開発環境

-   PHP
-   MAMP
